# Changelog

All notable changes to `@solana-builder-night/shared-types` are documented in this file.

The format follows Keep a Changelog and semantic versioning.

## [0.2.0] - 2026-04-21

### Changed
- Promoted shared-types package version from `0.1.0` to `0.2.0` for integration wave alignment.
- Updated semantic compatibility constraint to `@solana-builder-night/contracts@^0.2.0`.

### Notes
- This release is a compatibility lockstep update for contract freeze `v0.2.0` and does not introduce runtime type-breaking changes.

## [0.1.0] - 2026-04-21

### Added
- Initial shared type package for cross-service type consumption.
- Added canonical event, market, opportunity, execution, and risk type definitions.
- Declared semantic dependency constraint on `@solana-builder-night/contracts@^0.1.0`.
